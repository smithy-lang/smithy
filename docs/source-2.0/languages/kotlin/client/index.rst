=================
Client User Guide
=================

This guide walks through how to use `Smithy Kotlin <https://github.com/smithy-lang/smithy-kotlin>`_ to generate
Kotlin clients from a Smithy model of a service.

For a general overview of the Smithy IDL, see the Smithy :doc:`/quickstart` Guide

For a general introduction to the Smithy Kotlin framework, see the Smithy :doc:`../quickstart` Guide.
